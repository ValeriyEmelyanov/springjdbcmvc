#CRUD application with Spring MVC, Spring JDBC, Freemarker, MySql

Простое приложение, реализующее работу со списком пользователей.

#### Используемые технологии
* Spring MVC
* Spring JDBC
* Freemarker
* MySql
* Maven
* Tomcat
